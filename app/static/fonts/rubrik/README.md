# Rubrik font assets

Rubrik font binaries are not checked into source control. To serve the fonts locally, place the provided `.woff2` files in this directory using the following filenames:

- TPRubrik-ExtraLight.woff2
- TPRubrik-ExtraLightItalic.woff2
- TPRubrik-Light.woff2
- TPRubrik-LightItalic.woff2
- TPRubrik-Regular.woff2
- TPRubrik-RegularItalic.woff2
- TPRubrik-Medium.woff2
- TPRubrik-MediumItalic.woff2
- TPRubrik-SemiBold.woff2
- TPRubrik-SemiBoldItalic.woff2
- TPRubrik-Bold.woff2
- TPRubrik-BoldItalic.woff2
- TPRubrik-ExtraBold.woff2
- TPRubrik-ExtraBoldItalic.woff2

The CSS in `app/static/styles.css` expects these files to be available at runtime under `/static/fonts/rubrik/`.
